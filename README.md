# Freelancehunt AI Assistant Bot 🤖🚀

Профессиональный бот для мониторинга новых проектов на бирже **Freelancehunt** с интеллектуальной генерацией откликов через **Gemini AI**.

## 🌟 Основные возможности

- **Мгновенные уведомления**: Бот отслеживает новые проекты каждые 30 секунд и присылает их в выбранный канал/чат.
- **Интеллектуальные отклики**: Генерация персонализированных откликов на основе вашего портфолио.
- **Авто-скоринг**: ИИ анализирует проект и выставляет оценку сложности и релевантности (⭐⭐⭐⭐⭐).
- **Контроль языка**: Строгое соблюдение языка проекта (UA/RU) без смешивания.
- **Умные рекомендации**: Бот сам находит и вставляет подходящие ссылки на ваши работы из портфолио.
- **Автоматизация**: Автоматическая очистка старых проектов из базы данных.

## 🛠 Технологический стек

- **Язык**: Python 3.10+
- **Фреймворк**: [Aiogram 3.x](https://docs.aiogram.dev/) (полностью асинхронный)
- **ИИ**: Google Gemini Pro (flash-latest)
- **База данных**: SQLite + SQLAlchemy + aiosqlite
- **Настройки**: Pydantic-settings & Dotenv
- **Мониторинг**: Асинхронный цикл с использованием Tenacity (retry-логика)

## 📁 Структура проекта

```text
freelancehunt_bot/
```

## 🚀 Быстрый старт

### 1. Подготовка
Клонируйте репозиторий и создайте файл `.env` на основе шаблона:
```bash
cp .env.example .env
```

### 2. Настройка
Заполните `.env` своими данными:
- `FH_TOKEN`: Ваш токен API Freelancehunt.
- `BOT_TOKEN`: Токен вашего Telegram бота.
- `CHAT_ID`: ID канала или чата для уведомлений.
- `GEMINI_KEYS`: Ваши API ключи Gemini (через запятую).

### 3. Запуск
```bash
pip install -r requirements.txt
python main.py
```

## 🔒 Безопасность
Все конфиденциальные данные вынесены в файлы окружения. Файлы `.env` и `bot_data.db` добавлены в `.gitignore`.
